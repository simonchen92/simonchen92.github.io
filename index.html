Driving Me Bananas

My first full stack project where I created an application that tracks user's journal entries. A user can sign in to create a journal entry and they can clear and get all their entries. They can make updates to their journal entries or delete them. The application is built mainly using HTML, CSS/SCSS, JavaScript, jQuery, Handlebars, Ruby on Rails
Future Iteration: Possibly make more functionality and create a more journal entry-like application.

Survey Express

A team oriented full stack application I built along with 3 other developers. We created an application where a user can create, edit and update a suvey that is relative to theirs. A user can also take other user survey as well, but only once. The application is mainly built using HTML, CSS/SCSS, Bootstrap, JavaScript, MongoDB, Mongoose, Express.js, Chart.js, Toastr.

Future Iteration: I would want to let user create a survey with more than two options, make a real-time dashboard and make custom page routes so users can share links to their surveys
